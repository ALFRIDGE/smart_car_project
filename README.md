A modern Android application built with Jetpack Compose, MVI architecture, and offline-first data handling. The app allows users to browse, search, and view details about cars, with full support for loading, error, empty, and success UI states.

This project was built as a capstone to demonstrate clean architecture, state-driven UI, offline support, and unit testing with coverage reporting.

Features

Jetpack Compose UI (Material 3)

MVI Architecture (Actions → Reducer → State → UI)

Retrofit API integration

Offline support with Room

Search cars locally (Room-first), then API

Pull-to-refresh

4 UI states

Loading

Success

Empty

Error (with retry)

Unit tests with 80%+ coverage

JaCoCo coverage reports
